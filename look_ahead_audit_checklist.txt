[✓] 1. Entry timing — AMC vs BMO
    Rule: hour(MOSTIMPORTANTDATEUTC) >= 16 UTC → next NYSE trading day (AMC).
    Hour < 13 UTC → same-day close (BMO). Hour 13–15 UTC → next day (conservative).
    Implementation: assign_entry_date() in NB02, uses NYSE exchange_calendars.

[✓] 2. Forward returns are targets, never inputs
    fwd_1d/fwd_3d/fwd_5d/fwd_10d/fwd_20d computed from prices AFTER entry_date.
    All fwd_* columns excluded from FEATURE_COLS via NON_FEAT list in NB05.

[✓] 3. Cross-sectional features are point-in-time
    sector_pct_rank: sorted by entry_date; only prior trade-day events in denominator.
    Same-day earnings calls do not see each other's rank.
    sentences_sector_z: expanding mean/std with .shift(1) before .expanding().
    QoQ features: .shift(1) lag — current call never included in its own lag.

[✓] 4. Feature selection is part of training
    mutual_info_regression refit on training fold only at each walk-forward step.
    Top-50 features reselected each quarter — test fold never influences selection.

[✓] 5. Imputation and scaling are part of training
    StandardScaler.fit() on train fold; .transform() applied to test.
    fillna(0) applied identically to both train and test.

[✓] 6. Universe membership is point-in-time
    SP500/SP400/SP600: WRDS comp.idxcst_his (institutional subscription).
    GVKEY interval join — exact entry/removal dates, no ticker-rename errors.
    Caveat: current members only (thru=NULL); deleted stocks absent. Mild
    survivorship bias (<2% of events/year) documented in report Section 2.
    RU3K: CRSP crsp.msf top-3000 by market cap at each June reconstitution.
    Genuine PIT — no survivorship bias. 16 annual recons, 6,626 unique GVKEYs.

[✓] 7. No look-ahead in QoQ features
    qoq_delta = ATC_t - mean(ATC_{t-1:t-4}), all lags use .shift(1).
    qoq_acceleration = ATC_t - 2*ATC_{t-1} + ATC_{t-2}, all lagged.
    Current quarter score never appears in its own lag computation.

[✓] 8. Hyperparameter tuning on held-out period only
    Grid search on 2010–2017 only. Parameters frozen before any 2018+ data.
    Walk-forward test starts 2020-Q1. No re-tuning during test period.

[✓] 9. Placebo test passes
    Signal from Fluff/Filler-only AspectTheme columns → L/S Sharpe ≈ 0
    across all universes and horizons. Confirms no structural look-ahead.

[✓] 10. Corporate actions / delistings handled

[✓] 11. Fluff/Filler excluded from predictive features
