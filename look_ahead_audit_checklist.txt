[✓] 1. Entry timing — AMC vs BMO
    Rule: hour(MOSTIMPORTANTDATEUTC) ≥ 13 UTC → next NYSE trading day (conservative).
    Hour < 13 UTC → same-day close (BMO). Hours 13-15 UTC → next day (documented choice).
    Implementation: assign_entry_date() in notebook 02, uses NYSE calendar via
    pandas_market_calendars. All entry_date values are tz-naive trading-day closes.

[✓] 2. Forward returns are targets, never inputs
    fwd_1d / fwd_3d / fwd_5d / fwd_10d / fwd_20d computed from yfinance prices AFTER entry_date.
    Excluded from FEATURE_COLS by NON_FEAT list in notebooks 05 and 08.
    Note: feature names containing "fwd" (e.g. ks_bad_now_good_fwd) refer to "forward
    guidance" — an ATC Aspect dimension — NOT forward price returns.

[✓] 3. Cross-sectional features are point-in-time
    sector_pct_rank: expanding_pct_rank() loop starts at i=1; uses only series.iloc[:i].
    Sorted by SECTOR, entry_date (trade date, not call date) to avoid same-day leakage.
    sentences_sector_z: uses prior_expanding_z() with shift(1) before expanding mean/std
    so the current call is NOT included in its own normalisation denominator.
    QoQ features: qoq_delta = ATC_t − mean(ATC_{t-1}…ATC_{t-4}), using .shift(1).
    Current call never appears in its own lag calculation.

[✓] 4. Feature selection is part of training
    mutual_info_regression refit on training fold only at each walk-forward step.
    Top-50 features reselected each quarter — test fold never influences selection.

[✓] 5. Imputation and scaling are part of training
    StandardScaler.fit() on train fold; .transform() applied to test.
    Sector normalisation (NB08): sector means/stds fit on train, applied to test.

[✓] 6. Universe membership is point-in-time
    SP500: Wikipedia historical additions/removals reconstructed back to 1976.
    Verified: 458 unique tickers in 2010 → 503 by 2025 (genuine PIT growth, not current-only).
    SP1500 CAVEAT: Wikipedia SP400 history begins 2012, SP600 begins 2019.
    Pre-2012/2019 portions use current membership → mild survivorship. Reported alpha
    for those periods is an upper bound; flagged explicitly in results tables.
    RU3K CAVEAT: Proxy = all US-listed ATC tickers (~7,372 vs true ~3,000).
    Labelled "RU3K proxy" throughout. Reported alpha is an upper bound.

[✓] 7. No "next quarter minus current quarter" QoQ deltas
    All QoQ features use .shift(1): qoq_delta = ATC_t − mean(prior 4),
    qoq_acceleration = ATC_t − 2·ATC_{t-1} + ATC_{t-2}.
